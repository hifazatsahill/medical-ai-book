RAG Chatbot Iframe Embed Solution

This page demonstrates the iframe embed solution for the RAG Chatbot. The chatbot is loaded in an iframe below.

Embed code for HTML books:

<iframe 
    src="https://your-api-domain.com/embed/chatbot?collectionId=YOUR_COLLECTION_ID" 
    width="350" 
    height="500" 
    frameborder="0" 
    style="border-radius: 10px; box-shadow: 0 4px 6px rgba(0,0,0,0.1);">
</iframe>
